## Description

This is a fork of the original repo at https://github.com/fredemmott/StreamDeck-AudioSwitcher Check it out to see the original info.

Changes are as follows:

1. Added the ability to add an arbitrary amount of sound devices, rather than just one or two. It cycles through these devices
2. Added the ability to change both default and communication device at the same time

No support will be provided for this software. Has not been tested on MacOS. I will merge pull requests, however.
